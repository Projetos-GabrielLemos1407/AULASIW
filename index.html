Introdução ao HTML

DESENVOLVIMENTO PARA A INTERNET- HTML E CSS

-HTML(hyper Text Markup Language) é uma linguagem de Marcação de Hipertexto utilizada para criação de paginas na Web. Essa linguagem que o seu navegador (Browser) decodifica para exibir as páinas.

- o HTML é baseado no conceito de hipertexto.Hipertexto são conjuntos de elementos -ou nós- ligados por conexões. Estes elememtos podem ser palavras, imagens,videos,àudio,documetos e por ai mais.

Como funciona o HTML

Atraves de um documento HTML, ou seja com a extensão .html ou .htm., o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá-lo. Os arquivos .html podem ser visualizados em qualquer navegador (como Google Chrome, Safari, ou Mozilla Firefox).

geralmente é composto por diversas páginas HTML, como por exemplo: um website que contenha três páginas (uma homepage, uma página de contato e uma página de produtos) receberá ao menos três documentos .html distintos, sendo uma para cada página do website.

digo pode ser escrito através de qualquer editor de texto, como o próprio bloco de notas. Cada página consiste em uma série de tags (também chamados de elementos) que podem ser considerados os blocos de construção das páginas. Portanto, esses blocos são a maneira com a qual o HTML faz a marcação dos conteúdos, criando a hierarquia e a estrutura do mesmo, dividido entre seções, parágrafos, cabeçalhos, e outros.

  • HTML significa Hyper Text Markup Language
  • HTML é a linguagem de marcação padrão para a criação de páginas da web
  • HTML consiste em uma série de elementos
    • Os elementos HTML informam ao navegador como exibir o conteúdo Os elementos HTML identificam partes de conteúdo como "isto é um título", "isto é um parágrafo", "isto é um link", etc.

-Entenda o que é CSS

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível. Estes tipos de linguagem citados foram idealizados para serem funcionais e, por serem de marcação, precisaram ser aperfeiçoados. O CSS, no caso, veio como uma boa solução para aprimorar a estética dos sites. Por meio dele, é possível separar formatos de conteúdos, bem como definir como serão exibidos em cada página.

-O Que é e Para que Serve o W3C

W3C é uma organização internacional de padrões que desenvolve os pilares de tecnologias Web tais como HTML, CSS, SVG, XML e WCAG. O IDPF desenvolveu o padrão EPUB, utilizado globalmente em e-books acessíveis e em publicações digitais.

-Breve Historia do HTML

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.
